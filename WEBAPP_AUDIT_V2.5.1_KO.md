# Root Voca 웹앱 종합 점검 v2.5.1

## 점검 결론

v2.5 원본에서 정상 실행을 막는 결함 2건을 발견하고 수정했다.

1. 문서 하단의 초기 실행·로그인 코드가 `<script>` 밖에 위치했다.
   - 로그인 창이 열리지 않고 초기 실행 코드가 화면의 일반 문자열로 노출될 수 있었다.
   - 별도의 `<script>` 블록으로 복구했다.

2. 메일 오류 처리에 사용하는 `escapeHtml()` 함수가 정의되지 않았다.
   - 일부 수신자 실패, 메일 진단, 대용량 본문 축약 경로에서 JavaScript 오류가 발생할 수 있었다.
   - 공통 HTML 이스케이프 함수를 추가했다.

## 남아 있는 기능 차이

문제 우측의 6초 타이머는 현재 `6초 목표 표시`이다. 6초가 지나면 빨간색으로 강조되지만 자동 오답 처리되지는 않으며, 관리자가 제한시간을 변경하는 설정도 아직 없다. 이는 실행 오류가 아니라 초기 요구사항과의 기능 차이이므로 별도 보완이 필요하다.
